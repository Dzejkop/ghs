# ghs

A terminal UI for searching GitHub.

![gif](./repo/ghs.gif)

## Installation

```bash
cargo install ghs
```

## Usage

```bash
ghs
```

Set your GitHub token via environment variable:

```bash
export GITHUB_TOKEN=your_token_here
```

Or create a `.env` file with:

```
GITHUB_TOKEN=your_token_here
```
